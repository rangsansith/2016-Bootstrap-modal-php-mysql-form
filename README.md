# Responsive modal pop-up with php/mysql

The form will appear in the confirmation page in a shopping site.
[sample URL](http://www.hirokoymj.com/2015/vday2015/)

## Technologies
*Front-End*
- Bootstrap 3
- Bootstrap Modal.js
- jQuery 
- jQuery plug-in Masked Input

Back-End
- Apache 2.0
- PHP 5.5 (Using mysqli)
- MySQL 5.0.11
- phpMyAdmin 4.5.12
